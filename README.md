# shintobag

R package om met het BAG (/BRK/Top10 etc.) te werken, zoals adres validatie, database connecties,
links naar open bronnen (BAG Viewer, Google streetview etc.).

Wordt gebruikt in veel Shiny applicaties, data straten.
